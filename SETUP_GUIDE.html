TrueScore Setup Guide

Load your fake-review detector into Chrome in 3 simple steps

1
Open Chrome Extensions Page
Open a new Chrome tab and type the following URL into the address bar, then press Enter. This opens Chrome's extension manager where you can install local extensions.
// Paste this into Chrome's address bar:
chrome://extensions
What you'll see → find "Developer mode" in the top-right corner
chrome://extensions
Extensions
Developer mode
Your installed extensions appear here
💡 The Developer mode toggle must be turned ON (green) before you can load unpacked extensions. It's in the top-right corner of the page.
2
Click "Load Unpacked" → Select Your Folder
After enabling Developer mode, a new button called "Load unpacked" will appear in the top-left. Click it, then navigate to and select your truescore-extension folder (the one containing all the files you downloaded).
